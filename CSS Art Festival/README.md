# 🐼 CSS Art Festival — Pixel Panda

A cute panda character designed and rendered using only pure HTML5 and CSS3 (no JavaScript, SVGs, or external images).

## 🚀 Features
- **100% Pure CSS Art**: No external assets or SVGs are used. Every detail of the panda is constructed using HTML elements and CSS styling.

## 🛠️ Usage
No installation or build steps are required. Simply open `index.html` in any web browser to view the panda.
